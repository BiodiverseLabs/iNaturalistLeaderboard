# iNaturalist Leaderboard

## Overview
This project provides a leaderboard system for iNaturalist contributions, designed to encourage engagement and foster community competition. Users can view their standing and the top contributors in various categories.

## Features
- **User Rankings**: Displays user rankings based on contributions.
- **Categories**: Rankings are sortable by various categories, such as species observed or records submitted.
- **Easy Integration**: Can be integrated with an existing iNaturalist API.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/BiodiverseLabs/iNaturalistLeaderboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd iNaturalistLeaderboard
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```

Visit `http://localhost:3000` in your browser to view the leaderboard.
